### Projects

- **[Semtree](https://github.com/nikolasdehor/semtree)** - Semantic code trees for AI assistants. Indexes codebases with tree-sitter and provides token-optimized context to Claude Code, Cursor, Copilot. [PyPI](https://pypi.org/project/semtree/) · [Docs](https://nikolasdehor.github.io/semtree/)
- **[MCP Fiscal Brasil](https://github.com/nikolasdehor/mcp-fiscal-brasil)** - First MCP server for Brazil's fiscal system. Query CNPJ, NFe, SPED, eSocial. Published on [PyPI](https://pypi.org/project/mcp-fiscal-brasil/).
- **[Visual Eyes](https://github.com/nikolasdehor/visual-eyes)** - Claude Code plugin that captures screenshots, analyzes UI visually, and auto-fixes layout bugs.
- **[ShieldCode](https://github.com/nikolasdehor/shieldcode)** - Security hardening and error handling expertise for Claude Code. Prevents OWASP Top 10 vulnerabilities.
